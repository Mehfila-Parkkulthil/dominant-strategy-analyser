# Strategy dominance analyser

## Strictly dominated Strategy
A strategy is strictly dominated if there exists another strategy that gives a strictly higher payoff against every possible opponent strategy.

## Weakly dominated strategy
A strategy is weakly dominated if another strategy gives a greater or equal payoff in all cases, and strictly higher in at least one.

# Strategy Dominance Analyser

Analyses a two-player normal form game to identify strictly and weakly dominated strategies for each player.
What It Does:
- Takes a payoff matrix as input from both players
- Compares each strategy against all others for a given player
- Identifies which strategies are strictly or weakly dominated and by which strategy
